# Chatbot E-Commerce Menggunakan Multinomial Naive Bayes

This project implements a chatbot for e-commerce using the Multinomial Naive Bayes algorithm.

## Files
- `Chatbot_ML_MNB.ipynb`: Jupyter notebook containing the chatbot implementation.
- `e-commerce.json`: Dataset or configuration file for e-commerce data.

## Setup
1. Clone the repository.
2. Install required dependencies (e.g., scikit-learn, pandas, etc.).
3. Run the notebook to train and test the chatbot.

## Usage
Run the Jupyter notebook to interact with the chatbot.
